Для работы дашборда необходим портативный Node.js.
Следуйте инструкциям ниже для его установки.

═══════════════════════════════════════════════════════════════════

📥 ШАГ 1: СКАЧИВАНИЕ NODE.JS

1. Откройте браузер
2. Перейдите по ссылке: https://nodejs.org/dist/latest/
3. Найдите файл с названием: node-v[версия]-win-x64.zip
   Например: node-v20.10.0-win-x64.zip
4. Скачайте этот файл

═══════════════════════════════════════════════════════════════════

📦 ШАГ 2: РАСПАКОВКА

1. После скачивания найдите ZIP-архив в папке "Загрузки"
2. Щёлкните правой кнопкой мыши по архиву
3. Выберите "Извлечь всё..." или "Extract here"
4. Распакуйте архив

═══════════════════════════════════════════════════════════════════

📁 ШАГ 3: УСТАНОВКА В ПРОЕКТ

1. После распаковки вы получите папку с названием типа:
   node-v20.10.0-win-x64
   
2. Откройте эту папку

3. ПЕРЕИМЕНУЙТЕ эту папку в: node

4. ПЕРЕМЕСТИТЕ папку "node" в папку с дашбордом

5. Убедитесь, что структура выглядит так:

   Dashboard/
   │
   ├── node/                    ← Папка с Node.js
   │   ├── node.exe            ← Исполняемый файл Node.js
   │   ├── npm                 ← Менеджер пакетов (не используется)
   │   ├── npx                 
   │   └── ...
   │
   ├── dashboard.html
   ├── server.js
   ├── start.bat
   └── ваш_файл.xlsx

═══════════════════════════════════════════════════════════════════

✅ ШАГ 4: ПРОВЕРКА

1. Откройте папку "node"
2. Убедитесь, что внутри есть файл "node.exe"
3. Если файла нет, повторите шаги 1-3

═══════════════════════════════════════════════════════════════════

🚀 ШАГ 5: ЗАПУСК

1. Вернитесь в папку Dashboard
2. Двойной клик на файл "start.bat"
3. Если всё правильно, откроется консоль с сообщением о запуске сервера
4. Откройте браузер и перейдите по адресу: http://localhost:3000

═══════════════════════════════════════════════════════════════════

❓ ЧАСТЫЕ ВОПРОСЫ

Q: Нужны ли права администратора?
A: Нет! Портативный Node.js работает без прав администратора.

Q: Нужно ли устанавливать Node.js через установщик?
A: Нет! Используйте только портативную версию (ZIP-архив).

Q: Где скачать портативную версию?
A: Только на официальном сайте: https://nodejs.org/dist/latest/

Q: Какая версия Node.js нужна?
A: Любая актуальная версия (20.x или новее). Главное — скачайте файл
   с названием "win-x64.zip" (для Windows 64-bit).

Q: Можно ли использовать Node.js от другого проекта?
A: Да! Если у вас уже есть портативный Node.js, просто скопируйте
   папку "node" с файлом node.exe в папку с дашбордом.

═══════════════════════════════════════════════════════════════════

⚠️ ВАЖНО

• НЕ используйте глобально установленный Node.js (через установщик)
• Используйте ТОЛЬКО портативную версию (ZIP-архив)
• Папка ДОЛЖНА называться "node" (строчными буквами)
• Файл node.exe ДОЛЖЕН быть прямо в папке "node", а не в подпапках

═══════════════════════════════════════════════════════════════════

🆘 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

1. Проверьте, что папка "node" находится в папке Dashboard
2. Проверьте, что внутри папки "node" есть файл "node.exe"
3. Проверьте, что путь выглядит так:
   Dashboard\node\node.exe
   
4. Если проблема не решается, скачайте Node.js заново и повторите
   все шаги с начала
