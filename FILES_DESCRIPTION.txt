🚀 ОСНОВНЫЕ ФАЙЛЫ (Для работы дашборда)

├── dashboard.html
│   • Главный файл дашборда с двумя логиками (A и B)
│   • Открывается в браузере через сервер
│   • Содержит весь JavaScript, CSS и HTML код
│   • Размер: ~29 КБ
│   • Использовать: http://localhost:3000
│
├── server.js
│   • Node.js HTTP сервер
│   • Автоматически находит первый .xlsx файл
│   • Отдаёт dashboard.html и Excel данные
│   • Отключает кеширование
│   • Порт: 3000 (настраивается)
│
├── start.bat
│   • Batch файл для запуска сервера на Windows
│   • Проверяет наличие Node.js и файлов
│   • Запускает node\node.exe server.js
│   • Использование: Двойной клик
│
└── data.xlsx
    • Пример Excel файла с данными
    • Можно заменить на любой другой .xlsx файл
    • Дашборд автоматически найдёт первый Excel файл
